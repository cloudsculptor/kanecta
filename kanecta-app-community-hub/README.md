# Featherston Community Hub

A community information website for Featherston, New Zealand. Open source — all code and content is freely available.

🌐 [featherston.co.nz](https://featherston.co.nz)

## What it is

A simple website where locals can find information about events, transport, community groups, and more. Built and maintained by volunteers.

## Structure

```
featherston-client-web/   — the website (React, runs in the browser)
featherston-api/          — the backend server (Node.js)
```

## Tech

- **Website** — React, TypeScript, Material UI
- **API** — Node.js, Express
